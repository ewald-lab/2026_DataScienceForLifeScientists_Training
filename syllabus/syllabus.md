# Course Syllabus: Data Science for Life Scientists

## Course Information
- **Title**: Data Science for Life Scientists
- **Duration**: 3 afternoons
- **Target Audience**: Life scientists with basic computer skills
- **Prerequisites**: Basic understanding of biology, no programming experience required

## Learning Objectives
By the end of this course, participants will be able to:
- Collect and preprocess biological data using Python
- Train and evaluate simple machine learning models
- Apply deep learning techniques to biological datasets
- Interpret model results in the context of biological research
